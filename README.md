# blog

## Project setup
```
npm install
```

## json-server Setup
```
npm install -g json-server
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Download `data.json` file then move to the directory of `data.json` then run
```
json-server data.json
```

npm run build
